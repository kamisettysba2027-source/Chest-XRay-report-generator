3. MODEL ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

CNN ENCODER (Image Analysis):
  • Architecture: InceptionV3 (pre-trained on ImageNet)
  • Input: 256×256×3 RGB images (converted from grayscale)
  • Output: 2,048-dimensional feature vectors
  • Strategy: Transfer learning with frozen weights
  • Purpose: Extract medical image features

TEXT DECODER (Report Generation):
  • Embedding: 256-dimensional word embeddings
  • LSTM 1: 256 units, bidirectional context
  • LSTM 2: 256 units, semantic refinement
  • Dropout: 0.5 (regularization)
  • Output: Word predictions at each timestep

MERGE LAYER:
  • Method: Feature addition (image + text)
  • Purpose: Combine visual context with linguistic meaning

TOTAL PARAMETERS: 25,362,826

7. TECHNICAL IMPLEMENTATION
═══════════════════════════════════════════════════════════════════════════════

TRANSFER LEARNING:
  ✓ Used pre-trained ImageNet weights (InceptionV3)
  ✓ Froze CNN weights to preserve learned features

  ✓ Result: Better performance, faster training

MEMORY OPTIMIZATION:
  ✓ Implemented data generators (batch-by-batch loading)
  ✓ Used chunked processing (1,500 images per chunk)
  ✓ Result: Trained on Colab free tier (12GB RAM)

TRAINING OPTIMIZATION:
  ✓ Learning rate: 0.0005 (conservative decay)
  ✓ Optimizer: Adam
  ✓ Loss: Sparse categorical crossentropy

  ✓ Result: Prevented overfitting, efficient convergence
